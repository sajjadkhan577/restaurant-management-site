## 📖 Overview

**The Gastronomic Editorial** is a state-of-the-art, fully responsive web application tailored for high-end dining experiences. It features an immersive UI, dynamic menu exploration, live kitchen showcases, and user management capabilities, efficiently bridging the gap between luxury dining and digital accessibility.

## ✨ Features

- **Immersive Menus:** Extensive categorizations catering to diverse palettes (`BBQ`, `Burgers`, `Desserts`, and `Drinks`), highlighted by a stunning high-resolution aesthetic design.
- **Live Kitchen Showcase:** A dedicated experiential page bridging the visual gap via culinary choreography.
- **Authentication Flows:** Comprehensive `Sign Up` and `Sign In` capabilities nested smoothly inside of the navigation.
- **Order Tracking:** Track your ongoing orders seamlessly.
- **Profile Management:** Provide users with interactive dashboard zones to maintain settings and explore history.
- **Dark-Themed Palette:** Bespoke bespoke Tailwind architecture tailored specifically towards the luxury aesthetics of high-end gastronomy.
- **Fluid Responsiveness:** The ecosystem completely adapts interfaces, rendering appropriately across mobile, tablet, and widescreen environments via Tailwind CSS.

---

## 🛠️ Technologies Used

### Frontend Architecture
- **HTML5:** Semantic architecture layout positioning.
- **Tailwind CSS (CDN):** Fast, scalable, utility-first styling allowing rapid scaffolding.
- **JavaScript (Vanilla):** Pure logic components unburdened essentially by heavy external frameworks – handles Cart calculations, DOM rendering, and user flow popups natively.
- **Google Fonts & Material Symbols:** Clean modernized typography dynamically powered by `Inter` and `Manrope`, rounded out via Material Design icons.

## 💻 Local Setup Instructions

Want to run this project strictly on a local testing environment? Follow these core steps:

```bash
# 1. Clone the repository
git clone https://github.com/sajjadkhan577/restaurant-management-site.git

# 2. Navigate to the project directory
cd restaurant-management-site

# 3. Open the file natively inside your browser
# For UNIX / macOS:
open index.html

# For Linux:
xdg-open index.html

# For Windows:
start index.html
```

*(Alternatively, run a built-in HTTP server such as `python3 -m http.server 8080` to experience absolute routing execution natively!)*
